# 🎵 FNF-JAVA-CHART-EDITOR

Welcome to **FNF-JAVA-CHART-EDITOR**!

This is a standalone Friday Night Funkin' chart editor written entirely in **Java**.

Originally, this editor was planned to become the official chart editor for **Funkin Java**, but development on the main engine was taking longer than expected. Instead of waiting, I decided to release the chart editor separately.

---

# ✨ Features

- 🎹 Familiar layout inspired by **Psych Engine** and **JS Engine**
- 🚀 Designed to handle **millions of notes**
- 🎚️ Transparency slider to improve performance on extremely dense charts
- ⚡ Built with performance in mind

Compared to the JS Engine chart editor, this editor is designed to work with much larger charts while staying responsive.

---

# ☕ Installing Java

## Windows

```bash
winget install Microsoft.OpenJDK.21
```

## macOS (Homebrew)

```bash
brew install openjdk
```

## Ubuntu / Debian

```bash
sudo apt update
sudo apt install default-jdk
```

## Arch Linux / Manjaro / EndeavourOS

```bash
sudo pacman -S jdk-openjdk
```

Verify Java is installed:

```bash
java --version
```

---

# 🔨 Compiling

Compile the editor:

```bash
javac FNFChartEditor.java
```

Run the editor:

```bash
java FNFChartEditor
```

If everything launches successfully, you're ready to start charting!
